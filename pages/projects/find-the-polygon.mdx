# Find The Polygon

<p className="subtitle">A Game of Dimensions</p>

Find the Polygon is an game that . I created the game for the 2020 Luxembourg Tech School Game Development Challenge. The game received the 10th prize.

## About the Game

The aim of the game is to discover various 2D polygons hidden inside 3D polyhedra.
